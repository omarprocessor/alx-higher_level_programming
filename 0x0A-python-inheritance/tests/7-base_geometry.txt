**Nambari:**  
========  
**`integer_validator()`** inarudisha True ikiwa nambari ni integer.  

:: kwa nambari chanya  
   >>> BaseGeometry = __import__('7-base_geometry').BaseGeometry  
   >>> bg = BaseGeometry()  

:: na integer chanya  
   >>> bg.integer_validator("my_int", 12)  

:: na integer hasi  
   >>> bg.integer_validator("number", -12)  
   Traceback (most recent call last):  
   ValueError: nambari lazima iwe kubwa kuliko 0  

:: na hoja 3  
   >>> bg.integer_validator("numbers", 0, 1)  
   Traceback (most recent call last):  
   TypeError: integer_validator() inachukua hoja 3 za nafasi lakini 4 zimetolewa  

:: na integer isiyo na kipimo  
   >>> bg.integer_validator("infnumber", int('inf'))  
   Traceback (most recent call last):  
   ValueError: maandiko yasiyofaa kwa int() kwa msingi wa 10: 'inf'
